# Java Hangman
Graphical hangman built in Java using the Swing GUI framework. User has 5 tries to guess a word through letters in a fun, graphical way depicting a stickman's life dependent on guesses.

## Features
- Text field for letter input
- Buttons: Submit, Restart, Enter key support
- Labels displaying: Masked word, Letters guessed, Game messages
- Graphical hangman drawing
- Intuitively rejects already used letters, words, and ends game when hangman dead or user won, etc.

## Technologies Used
- Java
- Java Swing

## Controls
- Input one letter using keyboard
- Use Submit button or Enter key to enter a guess

## How To Run
- Clone the repository
- Compile Java files 'javac Hangman.java HangmanGraphic.java'
- Run program 'java Hangman'
